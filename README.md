# Custom time picker 
Demo is hosted at: https://bac-dam-1991.github.io/custom-time-picker/
## Instructions

### Run locally
1. Clone repository
2. Go to root of project
3. Run `npm i`
4. Run `npm start`

### Run via Docker
1. Clone repository
2. Go to root of project
3. Run `docker-compose up -d --build -V`
